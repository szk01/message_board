# SayHello

*Say hello to the world.*

Demo: http://sayhello.helloflask.com

![Screenshot](http://helloflask.com/screenshots/sayhello.png)

## Installation

```
$ git clone https://github.com/greyli/sayhello.git
$ cd sayhello
$ pipenv install --dev
$ pipenv shell
$ flask forge
$ flask run
* Running on http://127.0.0.1:8000/
```
